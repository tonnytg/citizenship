# citizenship

Citizenship (Cittadinanza Check) is a small React + TypeScript tool that helps people pré‑analisar a documentação para cidadania italiana. Tudo roda no navegador: você preenche dados da linhagem, faz upload de PDFs ou imagens, recebe uma pontuação com alertas e pode baixar um JSON local com o resumo.

## Onde fica o código
- `citizenbship-docs/`: aplicação React + Vite com Tailwind.

## Como rodar
```bash
cd citizenbship-docs
npm install
npm run dev   # servidor local
npm run build # build de produção
```
